# MMV Couture Platform

Plateforme de gestion complète pour ateliers de couture avec système CMS avancé.

## 🌟 Fonctionnalités

### 📸 **CMS Ultra-Avancé**
- **Positionnement d'images** : Gauche, droite, haut, centre, arrière-plan, superposition
- **Tailles d'images** : Petite, moyenne, grande, très grande, plein écran
- **Filtres et effets** : Noir et blanc, sépia, flou, luminosité, contraste, saturation
- **Animations** : 8 types (fondu, glissement, zoom, rebond, retournement, rotation, élastique)
- **Typographie avancée** : 6 polices, 6 tailles de titres
- **Effets au survol** : Zoom, panoramique, inclinaison, fondu, glissement

### 🎨 **Design Professionnel**
- **Images agrandies** : 20% plus larges, 60% plus hautes
- **Animations fluides** : 30+ animations CSS personnalisées
- **Responsive design** : Parfait sur tous écrans
- **Interface moderne** : Design épuré et professionnel

### 🛠️ **Gestion des Ateliers**
- **Dashboard admin** : Statistiques en temps réel
- **Gestion des artisans** : Profils complets
- **Système d'abonnements** : Paiements USSD intégrés
- **Showcase** : Galerie de créations

### 📱 **Multi-plateforme**
- **Application web** : Responsive et moderne
- **Application mobile** : Android via Capacitor
- **Desktop** : Application Electron

## 🚀 **Technologies**

- **Frontend** : React 19, TypeScript, TailwindCSS
- **Backend** : Node.js, Express
- **Database** : JSON local (extensible à MongoDB)
- **Mobile** : Capacitor, Android
- **Desktop** : Electron
- **Animations** : CSS personnalisées, Intersection Observer

## 📦 **Installation**

```bash
# Installer les dépendances
npm install

# Démarrer le serveur de développement
npm run dev

# Build pour production
npm run build

# Application desktop
npm run electron-dev

# Application Android
npm run android-sync
npm run android-build
```

## 🎯 **Utilisation du CMS**

1. **Accéder au dashboard admin**
2. **Cliquer sur "CMS"**
3. **Créer une nouvelle section**
4. **Uploader une image**
5. **Utiliser les options avancées** :
   - Positionner l'image (gauche/droite)
   - Choisir la taille (grande/très grande)
   - Appliquer des filtres
   - Ajouter des animations
6. **Sauvegarder et voir le résultat**
